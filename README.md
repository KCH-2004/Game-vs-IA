# PROJET IA: Implémentation du "Gomoku" avec la possibilité de jouer contre un autre joueur, ou bien de jouer contre une IA procédurale
## Présentation
Ce projet en Python propose une version du jeu du "Gomoku". L'objectif est double:
- Mode joueur: Affronter un autre humain ou une IA procédurale.
- Mode Performance : Mesurer et comparer l'efficacité des IA entre eux.

---

## Structure du projet: 

```
Game-vs-IA/
 .gitignore
 data.py
 game.py
 ia.py
 main.py
```

## Prérequis
L'implémentation étant fait en python, il est donc nécessaire d'avoir un interpréteur python 
Commandes pour installer python:

Linux (Debian/Ubuntu) 
```bash
sudo apt update && sudo apt install python3
```

Windows
```bash
winget install python.python.3
```

## Compilation et exécution

### Depuis un terminal
 1. Se placer à la racine du projet 
```bash
cd Game-vs-IA
```
 2. Exécuter le script
Linux (Debian/Ubuntu):
```bash
python3 main.py
```

Windows:
```bash
python main.py
```

### Depuis un IDE

1. Ouvrez le dossier parent chen_defoor dans votre IDE
2. Configurez l'interpréteur sur Python 3
3. Exécutez directement le fichier src/main.py
